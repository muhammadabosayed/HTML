Table

table

عبارة عن عمل جدول للبيانات ولكى يكون الجدول ظاهر يجب اضافة سمة border="1" كما يمكن أن نتحكم فى حجم الجدول عن طريق سمة width & height

caption

عبارة عن عمل عنوان ل table

tr

هى اختصار لمصطلح table row وهى عبارة عن عمل row للجدول

th

هى اختصار لمصطلح table head أى أنها تعبر عن row الرئيسى فى table وتكتب داخل tag ال tr

td

هى اختصار لمصطلح table data أى عبارة عن البيانات التى تكون داخل table وتكتب داخل tag ال tr

Output

colspan="رقم"

هذه السمة تضاف الى td او th لتقوم بدمج الخلايا مع بعض أفقيا على حسب الرقم تحدد عدد الخلايا التى تريد دمجها مع بعض فى table

Output

rowspan="رقم"

هذه السمة تضاف الى td او th لتقوم بدمج الخلايا مع بعض رأسيا على حسب الرقم تحدد عدد الخلايا التى تريد دمجها مع بعض فى table

Output

thead & tbody & tfoot

عبارة عن شكل منظم ل table بحيث row الخاص بالعنوان يكتب فى thead وباقى الصفوف التى هى عبارة عن البيانات التى سوف تخزن فى table تكتب فى tbody وفى حالة عمل footer للجدول نكتبه فى tfoot كآخر row فى الجدول

Output

Form

form

عبارة عن نموذج لتجميع بيانات المستخدم عن طريق إدخال بياناته فى مجموعة من الحقول وتحتوى على مجموعة من السمات الهامة

action

يحدد مسار الصفحة التى سوف ترسل البيانات لها

method

عبارة عن HTTP method تقوم بإخفاء البيانات أو اظهارها عند ارسالها فلو كانت method="post" فإن البيانات سوف تكون مخفية وأما اذا كانت method="get" فإن البيانات سوف تكون ظاهرة

novalidate

عبارة عن سمة boolean أى اذا قمنا بكتابة هذه السمة فإن form سوف تقوم بتجاهل كل حاجة مكتوبة غلط داخل حقول الادخال وكل عناصرها

autocomplete

هذه السمة تقوم بعمل تكملة للكلمات تلقائى فى حالة اذا كانت autocomplete="on" أما اذا كانت القيمة هى off فلن يتم عمل تكملة تلقائى للكلمات

label

عبارة عن كتابة نص يعبر عن ما سوف نكتبه داخل حقل الادخال ولديه سمة مهمة جداfor وهذه السمة تستخدم لربط هذا label مع radio او chekbox عن طريق اعطاء id ل inputوهذا ال id يكون هو قيمة السمة for

Output

br يستخدم لعمل سطر جديد

Input Types

type="text"

عبارة عن حقل إدخال والبيانات داخله تكون ظاهرة ونوعها string

type="password"

عبارة عن حقل إدخال خاص بكلمات المرور وأى شئ يكتب داخله يكون مشفر

type="submit"

هذا ال input يكون على هيئة زرار يقوم بعمل Submittingل data قبل ارسالها إلى Server

type="button"

عبارة عن زرار عادى ولكن استخدامه هو فعل أى شئ ولكن باستخدام JavaScript ويمكن أن نعطيه اسم عن طريق السمة value="..."

type="reset"

هذا الزرار يقوم بارجاع القيمة الافتراضية لكل الادوات الموجوده على ال form ويمكن تغيير اسمه عن طريق السمة value="..."

Output

type="radio"

عبارة عن تحديد خيار واحد فقط من عدة خيارات مختلفة ولكن يجب أن يكونوا كلهم لهم نفس الاسم الموجود فى السمة name="..." أما اذا كانوا ليس لهم نفس الاسم فإن كل واحده تكون مستقلة عن الاخرى

Output

type="checkbox"

عبارة عن اختيار اكثر من خيار من بين عدة خيارات مختلفة

Output

type="color"

هذا input يظهرلك color picker أى إنه يتعامل مع الالوان

type="data"

هذا input يظهرلك data اى يتعامل مع التاريخ

type="email"

هذا الحقل خاصة بإدخال إيميلات فقط

type="file"

هذا الزرار يسمح لك باختيار file من على جهازك ورفعه فى الموقع

type="number"

هذا الحقل خاص بالارقام فقط

type="search"

عبارة عن حقل مخصص للبحث

type="url"

هذا الحقل يأخد داخله رابط

textarea

وهى عبارة عن رسالة كبيرة نكتب فيها الكثير من الكلام وتحتوى على مجموعة من الخصائص منها rows=".." وهى عبارة عن القيمة الافتراضية لعدد صفوف هذه الرسالة عند تحميل الصفحة cols=".." وهى عبارة عن القيمة الافتراضية لعدد أعمدة هذه الرسالة عند تحميل الصفحة

Input Restrictions

checked

هذه السمة تجعل حقل checkbox او radio يكون selected عندما تحمل الصفحة

disabled

هذه السمة تجعل حقل الادخال للقراءة فقط كما يمكن أخذ القيمة المكتوبة copy ولكن لا يمكن ارسال هذه القيمة الى الخادم

readonly

هذه السمة تجعل حقل الادخال للقراءة فقط كما يمكن أخذ القيمة المكتوبة copy ولكن لا يمكن التعديل عليه والبيانات التى داخله= هذا الحقل يتم ارسالها الى الخادم

required

هذه السمة تجبرك على إدخال قيمة لهذا الحقل

placeholder

عبارة عن إظهار هذا الاسم فى حقل الادخال لمعرفة المستخدم ماذا سوف يكتب وعند بدء الكتابة داخل الحقل يتم مسح placeholder

maxlength

عبارة عن الحد الاقصى من الحروف التى تكون داخل هذا الحقل

minlength

عبارة عن الحد الادنى من الحروف التى تكون داخل هذا الحقل

max

عبارة عن أعلى قيمة فى حقل الادخال

min

عبارة عن أقل قيمة ممكنة فى حقل الادخال
